# 🦖 Dinosaur Game (Java)

A clone of the famous **Google Chrome Dinosaur Game**, developed in **Java using Swing**.  
The player controls a dinosaur that must avoid cacti to survive as long as possible.

# 🎮 Gameplay

* The dinosaur runs automatically.
* The player must **jump to avoid the cacti**.
* The score increases over time.
* A **collision with a cactus ends the game**.

---

# ⌨️ Controls

| Key | Action |
|-----|------|
| ↑ Up Arrow | Jump |
| Space | Jump |

---

# ⚙️ Technologies Used

* **Java**
* **Java Swing**
* **Timer (ActionListener)** for the game loop
* **Collision detection (AABB)**
